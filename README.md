![License](https://img.shields.io/badge/License-Unlicense-blue)
#
A Flappy Bird-inspired game built with Godot 4 using GDScript to explore physics, collision detection, and procedural obstacle spawning. 
# Features
- Procedural towering spawning
- Endless fun and interactive gameplay
- Optimized for smooth performance
# Installation
1. Head over to the [Releases](https://github.com/Shubiscoding/Dodge-The-Towers/releases/tag/DodgeTheTower) section.
2. Download the `.exe` file for your system.
3. Run the game and use the spacebar (or click) to play the game!
